# IGNITE-OMIS_Chatbot
An AI-powered chatbot system that helps users generate LinkedIn posts for promotions, achievements, events, or announcements. The bot collects user input, drafts a professional LinkedIn post, sends it to an admin for approval, and automatically publishes the approved post to LinkedIn.

## Features
- Chatbot interface for collecting post details from users
- AI-generated LinkedIn post drafts
- Admin approval workflow before publishing
- Automated LinkedIn posting
- Contributor leaderboard (planned feature)

## Contributing
1. Always create a new branch from the **main** branch before starting work on a feature or fix.
2. Make your changes and commit them to your feature branch.
3. Push your branch to GitHub.
4. Create a **Pull Request (PR)** to merge your branch into the **main** branch.
5. Your changes must be **reviewed and approved** before they are merged.

Direct commits to the **main** branch are not allowed. All changes must go through the Pull Request approval process.

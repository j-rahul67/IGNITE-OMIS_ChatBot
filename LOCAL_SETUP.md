# Local Development Setup Guide

This guide will help you set up and run the chatbot project locally on any machine.

## Prerequisites

1. **Node.js** (v18 or higher recommended)
   - Download and install from: https://nodejs.org/
2. **npm** (comes with Node.js)
3. **Git** (for cloning the repository)
   - Download and install from: https://git-scm.com/

## Steps to Run Locally

1. **Clone the repository**
   ```
   git clone <your-github-repo-url>
   cd <repo-folder>
   ```

2. **Navigate to the frontend directory**
   ```
   cd frontend
   ```

3. **Install dependencies**
   ```
   npm install
   ```

4. **Start the development server**
   ```
   npm run dev
   ```

5. **Open the app in your browser**
   - Visit the URL shown in the terminal (usually http://localhost:5173/)

## Notes
- No external backend or third-party services are required; all logic runs locally.
- If you add new dependencies or features, update this guide accordingly.
